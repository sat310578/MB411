Git ( / ɡɪt /​​ ⓘ \[ 8 ] ) — цесистемаконтролю версій \[ 9 ] , яка здатнакерувати версіямивихідного коду або даних. Вона часто використовується для контролювихідного кодупрограмістами,якіспільнорозробляютьпрограмне забезпечення.



Цілі розробки Git включають швидкість, цілісність даних та підтримку розподілених , нелінійних робочих процесів — тисяч паралельних гілок , що працюють на різних комп'ютерах. \[ 10 ] \[ 11 ] \[ 12 ]
